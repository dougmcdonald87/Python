# jens_adventure

A minimal starter Python project for "jens_adventure".

## What this repo contains

- src/jens_adventure: package source
- tests: pytest-based tests
- requirements.txt: development/test dependencies
- .github/workflows/python.yml: CI to run tests on Python 3.12

## Quickstart

Clone the repo and set up a virtual environment:

```bash
git clone git@github.com:dougmcdonald87/Python.git
cd Python
python -m venv .venv
source .venv/bin/activate   # on Windows: .venv\Scripts\Activate.ps1
pip install -r requirements.txt
pytest
```
